# 化妆品主页横幅图片说明

## 图片要求

请在 `frontend/public/images/` 文件夹中添加一个名为 `cosmetics-banner.jpg` 的图片文件。

### 推荐规格：
- **文件名**: `cosmetics-banner.jpg` 或 `cosmetics-banner.png`
- **尺寸**: 1200x400 像素（宽x高）
- **格式**: JPG 或 PNG
- **内容**: 化妆品相关的横幅图片（如化妆品产品展示、美容场景等）

### 如何添加：

1. 准备图片文件，命名为 `cosmetics-banner.jpg`
2. 将图片复制到 `frontend/public/images/` 文件夹
3. 刷新网页即可看到效果

### 当前状态：

如果图片不存在，横幅区域会自动隐藏，不影响页面其他功能。

### 图片建议：

- 化妆品产品展示
- 美容护肤场景
- 品牌宣传图片
- 促销活动横幅
